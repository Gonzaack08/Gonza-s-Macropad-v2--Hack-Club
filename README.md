Gonza's Macropad v2:



Second version of the hackpad, hope you like it.

Description:

A simple macropad for gaming, all 4 keycaps and the encoders have diferent characters combinations to use in discord or in game configuration.


## BOM

- 4× Cherry MX Switches
- 2× SK6812 MINI LEDs
- 1× XIAO RP2040
- 4× Blank DSA Keycaps
- 4× M3×16 Bolts
- 4× M3 Heatset Inserts
- 2x EC11 Rotary encoder
